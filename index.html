The Tool Tip Class

This week, we will be starting to take a very in-depth look at a class, to be more specific this will be a Tkinter programming class, that we will be creating for our Tool Tip, A class will enable us to easily create any type of tool tip that we want. We could consider this as the Tool Tip factory. Anytime we need a new type of tool tip, all we have to do is to call on this class and it will create it for us.
