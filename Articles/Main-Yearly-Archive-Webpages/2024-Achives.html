4 Working with all Browsers

This Week, we will be taking a look at how we can use Autoprefixer with Visual Studio to help us automatically add all of the prefixing to our CSS. This can be very helpful, especially since technology in side of the web designing world will change in a heartbeat, and when it does, all browsers have not quite caught up with what we are trying to do. This is where Autoprefixer can help. We can install it into our Visual Studio and it will take our CSS page and make all of the changes for us. In that way, we can work with the latest stuff, and Autoprefixer will fix things, where everything just works.
